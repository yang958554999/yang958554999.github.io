一、安装 PHP IntelliSense插件

打开vsode 编辑器，ctrl+shift+x 打开扩展商店，搜索 PHP IntelliSense 安装插件

二、配置 PHP IntelliSense插件，php.executablePath 路径（按自身情况填写）

OK，配置好了后重新vsCode 编辑器，试试效果吧，如果可以正常跳转那么恭喜你，如果还是无法跳转请看最后一步。
